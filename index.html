Crystal Collector Game

  • -You will be given a random "Target Number" at the start of the game.
  • -There are 4 crystals below. Each crystal will be given a randomized value at the start of each game.
  • -By clicking on the crystals, you will add that random value to "Your Score".
  • -The goal is to match "Your Score" with the "Target Score."
  • -If you match "Your Score" to the "Target Score" you win! The game will reset.
  • -If you surpass the "Target Score" with "Your Score", you lose, and the game resets.
